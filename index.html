cuba-vision-clinic

This project shows how your LightSwitch application could look after migration to the CUBA Platform, taking the official Microsoft Vision Clinic Sample as an example.

Migration is done automatically by the reverse-engineering functionality, provided by the CUBA Studio.

Find technical aspects of how to launch the application here.

Based on CUBA Platform 6.3.4
